# HTML Foundations 🏗️

Welcome to my HTML foundations repository! This project serves as a practical showcase of core HTML5 elements and web structuring concepts. 

## 🎯 Objective
The goal of this repository is to demonstrate a solid understanding of standard HTML elements, proper document structuring, and semantic web practices before moving on to CSS and JavaScript.

## 📂 What's Included

This repository contains standalone HTML files demonstrating the following concepts:

* **Semantic Tags:** Proper use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>` for better accessibility and SEO.
* **Forms:** Interactive user input fields including text boxes, radio buttons, checkboxes, dropdowns, and submit buttons.
* **Tables:** Structured data presentation using `<table>`, `<tr>`, `<th>`, and `<td>`.
* **Lists:** Both ordered (`<ol>`) and unordered (`<ul>`) lists, as well as nested lists.
* **Iframes:** Embedding external content (like videos or other web pages) using the `<iframe>` tag.
